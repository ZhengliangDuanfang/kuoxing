# 廓形

使用极少种类的构件线段，模拟中国传统木构建筑的大致轮廓与营造逻辑。使用单行命令的交互对构件进行操作。

注：本项目中的构件名称与中国传统木构建筑中的构件名称**并不重合**，搭建的具体依赖规则也有所差异。本项目仅尝试给出一种模拟大致形态的方法，对于中国传统木构建筑的实际情况，建议参考其它资料。

## 安装与运行方法

```shell
# 搭建环境
python -m venv venv
source venv/Scripts/activate # Git Bash
pip install -r requirements.txt

# 运行网页界面（推荐）
python main.py 8080 # 端口号可自定义，默认8080
# 运行命令行界面
python cli.py
```

## 命令操作语义规则
```
# 显示帮助
释
# 添加构件示例
置梁于柱甲寅至柱甲卯。内延二十寸。外延五十寸。
```

## 当前支持功能

- TXT文件批量导入、执行指令
- 构件
    - 柱：基本承重，垂直方向向上延申
        - 垂花柱：垂直方向向上、向下延申
    - 梁：水平方向连接、延申并承重
    - 栱：斜向向上延申并承重
    - 枋：水平方向连接
    - 檩：连接承重结构与屋顶结构
    - 点：屋顶结构参考点
    - 顶：屋顶结构表面
    - 脊：屋顶边缘轮廓
    - 墙：垂直于地面的建筑外表面
- 交互
    - 浏览器端图形显示界面，命令交互式更新
    - 命令行交互界面
    - PNG图片导出
    - 交互式视角调整、坐标显示设置
    - 构件颜色设置、透明度修改
- 自动生成构件编号注释
- 检查未利用构件

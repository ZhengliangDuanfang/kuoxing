# 廓形

使用极少种类的构件线段，模拟中国传统木构建筑的大致轮廓与营造逻辑。使用单行命令的交互对构件进行操作。

注：本项目中的构件名称与中国传统木构建筑中的构件名称**并不重合**，搭建的具体依赖规则也有所差异。本项目仅尝试给出一种模拟大致形态的方法，对于中国传统木构建筑的实际情况，建议参考其它资料。

```shell
# 搭建环境
python -m venv venv
source venv/Scripts/activate # Git Bash
pip install -r requirements.txt

# 运行网页界面（推荐）
python main.py 8080 # 端口号可自定义，默认8080
# 运行命令行界面
python cli.py
```
命令操作语义规则：
```
# 显示帮助
释
# 添加构件示例
置梁于柱甲寅至柱甲卯。内延二十寸。外延五十寸。
```
